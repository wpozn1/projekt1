
# 🎬 STREAMMATCH



**STREAMMATCH** is an application that lets you find movies for your personal preferences.

---

## ✨ Features

*   🔍 **Search**: Find movies that you like using *TMDB API* integration.
*   ⏱️ **Duration**: Set the duration of the movie you want to watch.
*   🎬 **Platforms**: Choose the platforms you have access to.
*   🎭**Genres**: Choose your favourite genres.
*   📖 **Your list**: Store the movies that you have already picked and watched.

---

## 🛠 Tech Stack

*   **Frontend**: [HTML](https://html.com/)
*   **Styling**: [CSS](https://css.com/)
*   **API**: [TMDB API](https://www.themoviedb.org/)
*   **State Management**: [JAVASCRIPT](https://www.javascript.com/)
*   **Backend** [DJANGO](https://www.djangoproject.com/)

---
